# FiBot Frontend

Frontend-only financial advisor experience implemented with React, TypeScript, and Tailwind CSS.

## Run

```bash
npm install
npm run dev
```

## Quality Gates

```bash
npm run lint
npm run build
npm run test
npm run test:e2e
```

## Key Features

- Sidebar with model selector and quick market insights
- Header with encryption and active-session indicators
- Chat timeline with assistant greeting and user send flow
- Input validation (prevents empty submits)
- Financial-advice disclaimer visibility
